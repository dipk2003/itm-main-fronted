# 🏪 Indian Trade Market — MVP Frontend

The frontend for Indian Trade Market (ITM), a B2B marketplace platform connecting Indian traders and businesses. This is the MVP deployment version built with Next.js and TypeScript.

![Vercel](https://img.shields.io/badge/Vercel-Deployed-black?logo=vercel&logoColor=white)

---

## ✨ Features

- **B2B Marketplace UI** — Browse, search, and connect with Indian traders
- **User Registration and Auth** — Secure onboarding and login
- **Product Catalog** — Browse and list products
- **Responsive Design** — Mobile-first with Tailwind CSS
- **Docker Support** — Containerized for easy deployment
- **Vercel Deployment** — Live at itm-main-fronted.vercel.app

## 🚀 Quick Start

```bash
git clone https://github.com/dipk2003/itm-main-fronted.git
cd itm-main-fronted
npm install
npm run dev
```

### Environment Setup

Configure the backend API endpoint in `.env.local`.
